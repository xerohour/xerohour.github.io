# Contributing to xerohour.github.io

Welcome, fellow traveler. If you're reading this, you've taken the red pill.

## 🐛 Bug Reports

Found a glitch in the Matrix? Please open an issue with:
- Description of the bug
- Steps to reproduce
- Expected behavior
- Screenshots (if applicable)
- Browser/OS info

## 💡 Feature Requests

Have an idea to enhance the construct? Submit a feature request:
- Clear description
- Use case
- Examples
- Alternative approaches considered

## 🔧 Pull Requests

1. Fork the repository
2. Create your feature branch (`git checkout -b feature/AmazingFeature`)
3. Commit your changes (`git commit -m 'Add some AmazingFeature'`)
4. Push to the branch (`git push origin feature/AmazingFeature`)
5. Open a Pull Request

## 📝 Code Style

- Follow existing Matrix theme aesthetics
- Keep animations performant (60fps)
- Test on multiple browsers
- Ensure responsive design

## 🎨 Theme Guidelines

- Primary color: `#00ff41` (Matrix Green)
- Background: `#000000` (Black)
- Font: 'Share Tech Mono', 'JetBrains Mono'
- Maintain cyberpunk aesthetic
